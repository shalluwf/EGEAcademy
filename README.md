# akademia_ege

Базовый каркас проекта на React + Vite.

## Запуск

```bash
npm install
npm run dev
```

## Дизайн‑токены

- `src/styles/tokens.css` — CSS variables (цвета из `res/specs/colors.txt` + базовая типографика Inter из `res/specs/fonts_*.txt`).
- `src/styles/global.css` — базовый reset + контейнер и демо‑стили.
